# 任务内容扩展与优先级交互优化 - 设计文档

**日期**: 2026-03-04
**状态**: 已批准
**类型**: 功能增强

---

## 1. 需求概述

### 1.1 功能目标

1. **任务内容扩展**：将任务从单一"标题"重构为"标题 + 内容"结构
2. **优先级交互优化**：改进优先级设置方式，提供更直观的交互体验

### 1.2 核心需求

- 任务支持标题和详细内容两个字段
- 首页同时展示标题和内容（最多2行，可展开）
- 优先级通过点击彩色圆点快速切换
- 编辑页使用分段控制器精确设置优先级
- 支持任务的展开/收起查看完整内容

---

## 2. 数据结构设计

### 2.1 类型定义

```typescript
// types/todo.ts
interface TodoItem {
  id: string
  title: string           // 标题（必填，最多50字）
  content: string         // 内容（可选，最多500字，默认空字符串）
  completed: boolean
  priority: 'low' | 'medium' | 'high'
  createdAt: number
  isExpanded?: boolean   // UI状态，不持久化
}
```

### 2.2 数据迁移策略

- 应用启动时自动检查并迁移旧数据
- 为没有 `content` 字段的任务添加默认值（空字符串）
- 迁移失败不影响应用正常启动

### 3.2 优先级色点组件

**文件**: `components/priority-dot/index.ts`

```typescript
Component({
  properties: {
    priority: String,  // 'low' | 'medium' | 'high'
    clickable: {
      type: Boolean,
      value: true
    }
  },

  methods: {
    onTap() {
      if (this.data.clickable) {
        this.triggerEvent('cycle', { id: this.data.todoId })
      }
    }
  }
})
```

**颜色映射**：
- `high` → 红色 (#FF3B30)
- `medium` → 蓝色 (#007AFF)
- `low` → 灰色 (#8E8E93)

## 5. 交互设计

### 5.1 展开/收起流程

1. 用户点击任务卡片中间区域（标题+内容）
2. 判断当前展开状态
3. 折叠状态 → 展开内容（显示完整内容）
4. 展开状态 → 收起内容（显示前2行）
5. 更新卡片高度（带动画过渡）
6. 展开新卡片时，自动收起其他已展开的卡片

### 5.2 优先级切换

**方式一：首页点击色点**
1. 用户点击优先级色点
2. 触发 `repository.cyclePriority(id)`
3. 优先级循环：低 → 中 → 高 → 低
4. 显示 Toast 提示："优先级：高"
5. 刷新列表，更新色点颜色

**方式二：编辑页分段控制器**
1. 用户在编辑页点击分段控制器
2. 高亮当前选中的优先级
3. 保存时写入数据
4. 返回首页，列表更新

### 5.3 点击区域规则

```
卡片点击分区：
├──────25%────│────60%────│──15%──┤
 切换完成状态    展开/收起    按钮区
```

- **左侧 25%**：切换任务完成状态
- **中间 60%**：展开/收起任务内容
- **右侧 15%**：编辑/删除按钮（左滑后显示）

## 6. 数据流与状态管理

### 6.1 首页状态

```typescript
data: {
  todos: [] as TodoItem[],
  expandedId: '' as string,  // 当前展开的任务ID
}
```

### 6.2 关键方法

```typescript
// 切换展开状态
toggleExpand(todoId: string) {
  const newExpandedId = this.data.expandedId === todoId ? '' : todoId
  this.setData({ expandedId: '' }, () => {
    if (newExpandedId) {
      this.setData({ expandedId: newExpandedId })
    }
  })
}

// 优先级切换（首页色点点击）
handlePriority(e: any) {
  const { id } = e.currentTarget.dataset
  const result = repository.cyclePriority(id)

  const priorityLabel = PRIORITY_CONFIG[result.priority].label
  wx.showToast({
    title: `优先级：${priorityLabel}`,
    icon: 'none',
    duration: 1000,
  })

  this.reloadTodos()
}
```

## 7. 错误处理

### 7.1 输入验证

- **标题**：必填，最多50字
- **内容**：可选，最多500字
- 空内容不显示内容行，保持卡片高度一致

### 7.2 数据迁移

应用启动时自动检查并迁移旧数据，失败不影响启动。

### 7.3 展开状态异常

确保同一时间只有一个卡片展开，避免布局混乱。
